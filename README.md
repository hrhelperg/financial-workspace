# Financial Workspace

Financial Workspace is a structured financial workspace for freelancers and small businesses.

It helps users manage invoices, expenses, clients, cashflow, documents, and payment reminders in one place.

## Core modules

- Invoice generator
- Client management
- Expense tracking
- Cashflow overview
- Document storage
- Payment reminders
- Automation layer

## Product direction

The goal is not to build another invoice tool, but a system that helps freelancers and small businesses get paid faster, understand cashflow, and organize financial operations.

## Initial stack

- Next.js
- TypeScript
- PostgreSQL
- Drizzle ORM
- Tailwind CSS
- Inngest
- Postmark
- Cloudflare R2
